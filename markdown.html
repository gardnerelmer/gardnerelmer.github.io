Markdown both the name of "a plain text formatting syntax" or lightweight markup language and its a software tool for converting that text to HTML.

Blockquote

> character prepended to the text 
	>Placeholder Text line 1
	Placeholder Text line 2
	Placeholder Text line 3
Becomes:
	<blockquote>
	  <p>Placeholder Text line 1
	Placeholder Text line 2
	Placeholder Text line 3</p>
	</blockquote>

Code Block

Prepend four spaces or a tab to the text 
	    Place Holder Text line 1
		Place Holder Text line 2
	    Place Holder Text line 3
 Becomes:
	<pre><code>Place Holder Text line 1
	Place Holder Text line 2
	Place Holder Text line 3
	</code></pre>

Code Span

Enclose text within two ` characters, this is a grave accent aka backquote, or backtick.
(Note: Not to be confused with ' the single quote) 
	`Placeholder Text` line 1
	Placeholder `Text line 2`
	Placeholder Text `line` 3
Becomes:
	<p><code>Placeholder Text</code> line 1
	Placeholder <code>Text line 2</code>
	Placeholder Text <code>line</code> 3</p>

Emphasis

Enclose text within one or two *
Enclose text within one or two _ 
	*Placeholder Text line 1*
	**Placeholder Text line 2**
	_Placeholder Text line 3_
	__Placeholder Text line 4__
Becomes:
	<p><em>Placeholder Text line 1</em>
	<strong>Placeholder Text line 2</strong>
	<em>Placeholder Text line 3</em>
	<strong>Placeholder Text line 4</strong></p>
