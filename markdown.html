Markdown both the name of "a plain text formatting syntax" or lightweight markup language and its a software tool for converting that text to HTML.
(Note: \ is the escape character for \`*_{}[]()#+-.!)

Blockquote

> character prepended to the text 
	>Placeholder Text line 1
	Placeholder Text line 2
	Placeholder Text line 3
Becomes:
	<blockquote>
	  Placeholder Text line 1
	Placeholder Text line 2
	Placeholder Text line 3
	</blockquote>

Code Block

Prepend four spaces or a tab to the text 
	    Place Holder Text line 1
		Place Holder Text line 2
	    Place Holder Text line 3
 Becomes:
	<pre><code>Place Holder Text line 1
	Place Holder Text line 2
	Place Holder Text line 3
	</code></pre>

Code Span

Enclose text within two ` characters, this is a grave accent aka backquote, or backtick.
(Note: Not to be confused with ' the single quote) 
	`Placeholder Text` line 1
	Placeholder `Text line 2`
	Placeholder Text `line` 3
Becomes:
	<code>Placeholder Text</code> line 1
	Placeholder <code>Text line 2</code>
	Placeholder Text <code>line</code> 3

Emphasis

Enclose text within one or two *
Enclose text within one or two _ 
	*Placeholder Text line 1*
	**Placeholder Text line 2**
	_Placeholder Text line 3_
	__Placeholder Text line 4__
Becomes: 
	<em>Placeholder Text line 1</em>
	<strong>Placeholder Text line 2</strong>
	<em>Placeholder Text line 3</em>
	<strong>Placeholder Text line 4</strong>

Headings

# character prepended to line = or - characters directly below lines for h1 and h2 
	Placeholder Text line 1a
	=
	#Placeholder Text line 1b
	Placeholder Text line 2a
	-
	##Placeholder Text line 2b
	###Placeholder Text line 3
	####Placeholder Text line 4
	#####Placeholder Text line 5
	######Placeholder Text line 6
Becomes:
	<h1>Placeholder Text line 1a</h1>

	<h1>Placeholder Text line 1b</h1>

	<h2>Placeholder Text line 2a</h2>

	<h2>Placeholder Text line 2b</h2>

	<h3>Placeholder Text line 3</h3>

	<h4>Placeholder Text line 4</h4>

	<h5>Placeholder Text line 5</h5>

	<h6>Placeholder Text line 6</h6>

Horizontal Rule

Three or more of the - hyphens, * asterisks, _ underscore character 
	Placeholder Text line 1

	_ __

	*****

	- - -

	Placeholder Text line 2
Becomes:
	Placeholder Text line 1

	<hr />

	<hr />

	<hr />

	Placeholder Text line 2

Links

URL or E-mail address inside < > (Note: E-mail addresses will be obfuscated with Hex and Decimal encoding)
Link text in [ ], followed by : and URL, optional Title between " "
Link text in [ ], followed by reference inside of [ ]
Prepend [ ] with ! to change from hyperlink to image

References
Reference name between [ ] followed : and URL, optional Title between " "
Implicit reference has no name, URL between [ ], followed by empty []
	[Ex1]: http://www.example.com "Reference"
	[Ex2]: image.png "Image, Implicit Reference"

	<http://www.example.com>
	<address@example.com>  
	[Inline Link](http://www.example.com "Inline")  
	[Reference Link][Ex1]  
	![Ex2][]
Becomes:
	<a href="http://www.example.com">http://www.example.com</a>
	<a href="mailto:address@example.com">address@example.com</a> <br />
	<a href="http://www.example.com" title="Inline">Inline Link</a> <br />
	<a href="http://www.example.com" title="Reference">Reference Link</a> <br />
	<img src="image.png" alt="Ex2" title="Image, Implicit Reference" />

List

Prepend line with + or - or * for unordered list
Prepend line with number-period-space sequence for ordered list
(Note: any single digit number, not just 0 or 1) 
	1. Placeholder Text line 1
	+ Placeholder Text line 2
	- Placeholder Text line 3
	* Placeholder Text line 4
Becomes:
	<ol>
	<li>Placeholder Text line 1</li>
	<li>Placeholder Text line 2</li>
	<li>Placeholder Text line 3</li>
	<li>Placeholder Text line 4</li>
	</ol>

Paragraphs and Line Breaks

Paragraphs are one or more consecutive lines of text
Append two or more spaces to the end of a line for a line break
	Placeholder Text line 1
	Placeholder Text line 2  
	Placeholder Text line 3

	Placeholder Text line 4
Becomes:
	<p>Placeholder Text line 1
	Placeholder Text line 2 <br />
	Placeholder Text line 3</p>

	<p>Placeholder Text line 4</p>
